# Project Plan

## Project Title
AI Career Intelligence Platform

## Problem Statement
Students and early-career professionals often struggle to understand which career roles match their current skills, what skills they need to improve, and how to make their resume more targeted for job applications.

## Objective
The goal of this project is to build a career intelligence dashboard that analyzes a user's skills, resume text, and job description to generate role-fit scores, skill gaps, portfolio project ideas, learning plans, and a 30-day action roadmap.

## Target Users
- Students
- Freshers
- Early-career professionals
- Career switchers

## Core Features
- Career profile form
- Role match score
- Career readiness percentage
- Skill gap analysis
- Portfolio project recommendations
- Resume strength analyzer
- Job description matcher
- Learning plan suggestions
- Interview preparation prompts
- 30-day career roadmap
- Save and export career reports

## User Flow
1. User enters career level, target role, skills, and goal.
2. Platform generates a career readiness score.
3. Platform shows matching roles and missing skills.
4. User pastes resume text to get resume improvement suggestions.
5. User pastes a job description to compare against their skills.
6. User saves or exports the final career report.

## Tech Stack
- HTML
- CSS
- JavaScript
- Browser LocalStorage

## Current Scope
This version is a frontend MVP. It uses rule-based logic to simulate AI-style recommendations and career analysis.

## Future Improvements
- Add real AI report generation using OpenAI API
- Add user login
- Store reports in a database
- Add resume PDF upload
- Add charts and analytics
- Convert project to Next.js and TypeScript
- Deploy on Vercel or Netlify

## Resume Value
This project demonstrates frontend development, product thinking, user workflow design, recommendation logic, and career-focused AI application design.
