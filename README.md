# AI Career Intelligence Platform

An AI-style career planning dashboard that analyzes a user's skills, resume text, and job description to generate role-fit scores, skill gaps, portfolio project ideas, learning plans, and a 30-day action roadmap.

## Problem

Students and early-career professionals often struggle to understand which roles fit their skills, what gaps to close, and how to make their resumes more targeted.

## Solution

CareerIntel AI gives users a structured career report from three inputs:

- Career profile
- Resume text
- Job description

The app turns those inputs into practical recommendations that a learner can act on immediately.

## Features

- Career profile form
- Role match scoring
- Career readiness percentage
- Skill gap recommendations
- Portfolio project suggestions
- Learning plan generation
- Interview story prompts
- Resume strength analyzer
- Job description matcher
- Saved reports using browser local storage
- Exportable career report
- Fully responsive dashboard UI

## Tech stack

- HTML
- CSS
- JavaScript
- Browser LocalStorage

## How to run

Open `index.html` in your browser.

No installation is required.

## Project files
